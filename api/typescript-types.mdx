Svelte Drawer exports TypeScript types for all component props, enabling type-safe usage and better IDE autocomplete support.

## Importing types

```typescript
import type {
  DrawerProps,
  DrawerContentProps,
  DrawerOverlayProps,
  DrawerHandleProps,
  DrawerVariant,
  DrawerVariantsProps,
  DrawerPortalProps,
  DrawerHeaderProps,
  DrawerFooterProps,
} from 'svelte-drawer';
```

## Type definitions

### DrawerProps

Props for the main `Drawer` component.

```typescript
interface DrawerProps {
  open?: boolean;
  onOpenChange?: (open: boolean) => void;
  direction?: "bottom" | "top" | "left" | "right";
  closeOnEscape?: boolean;
  snapPoints?: number[];
  activeSnapPoint?: number;
  onSnapPointChange?: (snapPoint: number) => void;
  portal?: boolean;
  portalContainer?: HTMLElement | string;
  persistState?: boolean;
  persistKey?: string;
  persistSnapPoint?: boolean;
}
```

**Use when:** Defining props for the root Drawer component or creating wrapper components.

**Example:**

```svelte
<script lang="ts">
  import { Drawer } from 'svelte-drawer';
  import type { DrawerProps } from 'svelte-drawer';
  
  let props: Partial<DrawerProps> = {
    direction: 'bottom',
    snapPoints: [0.5, 0.75, 1],
    closeOnEscape: true,
  };
</script>

<Drawer {...props}>
  <!-- content -->
</Drawer>
```

### DrawerVariant

Union type for predefined drawer style variants.

```typescript
type DrawerVariant = "default" | "sheet" | "dialog" | "minimal" | "sidebar";
```

**Use when:** Specifying a variant for the `DrawerVariants` component or creating type-safe variant selection.

**Example:**

```svelte
<script lang="ts">
  import type { DrawerVariant } from 'svelte-drawer';
  
  let selectedVariant: DrawerVariant = 'sheet';
  
  function changeVariant(variant: DrawerVariant) {
    selectedVariant = variant;
  }
</script>
```

### DrawerVariantsProps

interface DrawerVariantsProps {

## Creating typed wrappers

You can use these types to create strongly-typed wrapper components:

```svelte
<script lang="ts">
  import { Drawer } from 'svelte-drawer';
  import type { DrawerProps } from 'svelte-drawer';
  
  interface CustomDrawerProps extends DrawerProps {
    customProp?: string;
  }
  
  export let customProp: string | undefined = undefined;
  export let open: boolean = false;
  export let direction: DrawerProps['direction'] = 'bottom';
  
  // Rest of your props
</script>

<Drawer {open} {direction} {...$$restProps}>
  <slot />
</Drawer>
```
