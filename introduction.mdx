Svelte Drawer is a powerful, accessible drawer component for Svelte 5, inspired by [Vaul](https://github.com/emilkowalski/vaul). It provides a smooth, gesture-driven interface with comprehensive support for mobile touch interactions and keyboard navigation.

## Key Features

<AccordionGroup>
  <Accordion title="Animations & Interactions">
    - Smooth animations with gesture-driven dragging (mouse & touch)
    - Mobile-optimized drag handling with scroll prevention
    - Snap points for iOS-like multi-height drawers

  <Accordion title="Flexible Positioning">

  <Accordion title="Developer Experience">

    - Nested drawer support
    - Full TypeScript support

  </Accordion>
  
  <Accordion title="Accessibility">
    - Keyboard shortcuts (Escape to close, Tab navigation)

    - Screen reader friendly
  </Accordion>

## What Makes It Special?

**Gesture-Driven Design**: Unlike traditional modal dialogs, Svelte Drawer responds naturally to user gestures. Drag down to dismiss, drag to snap points, or use keyboard shortcuts - it works the way users expect.

**Mobile First**: Built with mobile in mind from the ground up. Touch interactions are smooth, scroll prevention works reliably, and the component feels native on mobile devices.

**Flexible & Composable**: Use the low-level `Drawer`, `DrawerOverlay`, and `DrawerContent` components for full control, or leverage prebuilt variants and optional header/footer components for rapid development.
